# ?? DeepSeek 引用链接提取器

一个基于 Streamlit 和 Playwright 的自动化工具，用于批量提取 DeepSeek 对话中的引用来源并生成分享链接。

## ? 功能特点

- ?? **批量处理**：支持一次性输入多个问题，自动批量处理
- ?? **自动登录**：首次手动登录后自动保存登录状态，后续无需重复登录
- ?? **中英文自动适配**：自动识别界面语言，支持中英文按钮
- ?? **引用提取**：自动捕获 AI 回答中的引用来源（标题、网站、URL）
- ?? **分享链接生成**：自动生成每个对话的分享链接
- ?? **数据导出**：支持导出 CSV 格式的引用数据
- ?? **实时日志**：实时显示运行日志，方便追踪进度

## ??? 技术栈

- **Streamlit**：Web 界面框架
- **Playwright**：浏览器自动化
- **Pandas**：数据处理和导出
- **Python 3.11+**：开发语言
